## 🚀 Quick Start Guide

### Prerequisites
- **Node.js** (v16+ recommended)
- **MongoDB** (local or cloud instance)
- **Git** for version control

### 🔧 Installation

<details>
<summary><b>🖥️ Backend Setup</b></summary>

```powershell
# Navigate to backend directory
cd BACKEND

# Install dependencies
npm install

# Create environment file
echo "# Environment Variables" > .env
echo "MONGODB_URI=mongodb://localhost:27017/figgz_restaurant" >> .env
echo "JWT_SECRET=your_super_secret_jwt_key_here" >> .env
echo "CLIENT_URL=http://localhost:5173" >> .env
echo "PORT=3000" >> .env

# Start development server
npm run dev
```

</details>

<details>
<summary><b>🎨 Frontend Setup</b></summary>

```powershell
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start development server
npm run start
```

</details>

### 🌐 Access Your Application

| Service | URL | Description |
|---------|-----|-------------|
| 🎨 **Frontend** | http://localhost:5173 | Customer-facing website |
| ⚙️ **Backend API** | http://localhost:3000 | RESTful API endpoints |
| 📊 **Admin Dashboard** | http://localhost:5173/admin | Management interface |
